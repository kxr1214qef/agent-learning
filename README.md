## 学习节奏

- 共 9 周
- 每周学习 6 天，休息 1 天
- 每天 1 小时
- 每天完成一次记录
- 每次学习尽量提交一个小 commit

## 使用方式

1. 阅读 `LEARNING_PLAN.md`，找到当天任务。
2. 在 `notes/` 中记录概念、命令、代码片段和心得。
3. 在 `progress/week-xx.md` 中完成当天打卡。
4. 完成实操后，视情况提交 commit，并记录 commit 链接。
